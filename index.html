Anagram Classification October 2017

The goal of this project was to take an input file that contains one word per line and write an output file that places anagram equivalence classes on the same line. The project challenge was that no outside libraries were allowed for data structures, sorting, and other topics covered in class. The project also had to run very efficiently. I was able to complete the project using a HashMap that forced anagram collisions. My runtime for a data file with over 6,000,000 entries was less than 14 seconds.
